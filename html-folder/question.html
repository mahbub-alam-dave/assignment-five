Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. For example, an element can be accessed by ID, by Class, by TagName, by querySelector ad querySelectorAll etc. Although, there are some more ways to select and element in the DOM, these five (ID, class, tagName, querySelector and querySelectorAll) are the best and recommended way to get and Element in the DOM

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHtml returns, sets and parses contents including html tags. innerText returns and sets only the visible text content of an element. It doesn’t show hidden text of the element that is set to display hidden. Also, innerTest preserves spacing based on how it’s shown in the browser. TextContent returns and sets all the content of an element whether the element is set to display hidden. It doesn’t parse html elements. Additionally, innertext doesn’t preserve spacing.

Question-3: What is event delegation in the DOM?

Event delegation is a very useful method in javascript where we can use a single event listener to a parent element instead of using multiple event listeners to individual child elements.

Question-4: What is event bubbling in the DOM?

When an event starts on an element and then moves up through its parent elements until it reaches the top of the document, this process is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

By using document.createElement() method I can create an element. Then, to add the created element to the document I need to use append() or appendChild() method. Lastly, to remove elements I need to use remove() or removeChild() method.
